Initializes I2C instance associating GPIOs to I2C signals.

Parameters
SDAGPIOPin to use for SDA signal
SCLGPIOPin to use for SCK signal

Example: 

// 4 = SDA, 15 = SCL (WARN: disconnect VGA connector!!)
I2C.begin(GPIO_NUM_4, GPIO_NUM_15);
