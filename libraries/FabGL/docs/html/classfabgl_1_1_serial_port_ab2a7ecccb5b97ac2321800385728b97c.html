Associates GPIOs to serial port signals.

This method must be called before setup(). setSignals() can be called just one time.

Parameters
rxPinUART RX pin GPIO number.
txPinUART TX pin GPIO number.
rtsPinRTS signal GPIO number (-1 = not used)
ctsPinCTS signal GPIO number (-1 = not used)
dtrPinDTR signal GPIO number (-1 = not used)
dsrPinDSR signal GPIO number (-1 = not used)
dcdPinDCD signal GPIO number (-1 = not used)
riPinRI signal GPIO number (-1 = not used)

Example: 

serialPort.setSignals(34, 2);  // rx = GPIO 34, tx = GPIO 2
serialPort.setup(2, 115200, 8, 'N', 1, FlowControl::Software);
